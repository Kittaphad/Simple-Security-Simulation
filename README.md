High-Security Passcode Simulation (Python)

โปรแกรมจำลองระบบความปลอดภัยที่เน้นความซับซ้อนของรหัสผ่านและการตอบโต้การบุกรุก

ฟีเจอร์หลัก
**Complex Password Generation:** สุ่มรหัส 6 หลัก (A-Z, a-z, 0-9) ความน่าจะเป็น 5.6 หมื่นล้านรูปแบบ
**Intrusion Detection:** จำกัดการเข้าถึงเพียง 3 ครั้ง
**Emergency Self-Destruct:** ระบบนับถอยหลังทำลายข้อมูลเมื่อถูกบุกรุก

ภาษาที่ใช้
- Python 3.x
- Libraries: `random`, `time`, `string`
